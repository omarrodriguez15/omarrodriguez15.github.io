Arduino Arduino is a simple to use and setup microcontroller. Arduino is not to be confused with devices like the Raspberry Pi although they can achieve the same goals they are diffent types of systems. The Pi can run an OS like linux. Arduino you compile some simple C like code into a program and upload it to an arduino board and it will run the program in a infinite loop.

You can directly control IO pins that range from digital, analog, and PWM(Pulse Width Modulation) There a variety of controller options in different sizes and computing capablities. The most popular in my opinion and my favorite is the UNO

The Arduino is typically programmed using the Arduino IDE and special programming language based largely off of the Open Source programming language Processing. There are two main functions for a typical Arduino program. A setup function and a loop function. The setup function is where you would initilize variables and set 'pinmodes' for the IO pins on the board and things like that. The loop function is continously executed until the devie is either reset or loses power.

The arduino boards can be stacked with devices called shields. The shields essentially extend the capability of the microcontroller. For example a common shield used is an ethernet shield which gives the arduino the capability to be connected to a network or the Internet. Contact Me
