IDE's (Integrated Development Enviornment)

 An IDE is basically a really fancy text editor. IDE's usually have a lot of common functionallity such as Syntax highlighting, auto completion, built in debuggers, dependency managers, template generators, just to name a few.

Some of the most popular IDE's are Visual Studio, Xamarin Studio, Android Studio, Eclipse, and Xcode. Each one of these IDE's have a main framework and or language that they are specifically good at but are able to handle other languages or frameworks.

Visual Studio

Visual Studio, personally my favorite, is a Microsoft product that as of today only runs on Windows. Its supports quite a few languages such as C#, C++, F#, VB, python. It also supports some other languages just not super well yet, such as Javascript, HTML, and Typescript.

Visual Studio works best with C#, VB, F#, and C++. Basically languages that implement .Net Framework. The downside is it only runs on Windows and it is expensive. However you can get a free version that is limited and good enough for small scale projects. There is a new Text editor/Light weight IDE called Visual Code that is cross platform, meaning it will work on OSX or Linux. It looks very promising and is surprisingly powerful and lightweight. This IDE is also very big and can be laggy at times. That being said it is still my favorite IDE.

Xamarin Studio

The offical IDE for Xamarin projects. Was originally on both Windows and OSX but when Microsoft bought the company it was wrapped up into Visual Studio. Now Xamarin projects are developed in Visual studio on Windows but still use Xamarin studio on OSX. Xamarin projects are cross platform mobile and desktop applications. All projects use the C# language.

Android Studio

An IDE created by Google. The IDE works on OSX and Windows. The IDE is fairly young but has recieved widespred adoption among Native Android developers. It is currently the only officially supported IDE (I think).

Eclipse

Eclipse is the go to IDE for Java delvelopers. It works on OSX and Windows. It was originally the go to IDE for Android apps until recently when Google created Android Studio.

Xcode

Xcode is the main IDE for IOS and OSX App development. It is free on Mac's however you need a developers license (around $100 a year) to actually build and deploy to the app store. The IDE only works on OSX. It supports Objective-C and Swift. I have really used it passed hello world apps so I don't have much to say about it.

Glossary
Syntax highlighting - Color text different depending on what category it falls into to more easily read and understand code.
Auto completion - Similar to normal text auto completion but targeted to language specific terms, methods, or functions etc.
Debugger - used to run and test a program in an Enviornment where you can examine the code dynamically.
Dependency Manager - Handles downloading and updating external packages.
Template generator - Creates new projects from scratch. Creates all the folders and files needed for simple hello world projects.
